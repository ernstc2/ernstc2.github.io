Malung - COVID-19 Classifier

Malung is a cutting-edge COVID-19 classification tool designed to assist healthcare professionals in identifying COVID-19 from CT scans. It uses advanced machine learning models to process patient data and provide real-time predictions.

Features:

  • Accurate classification of CT scan images.
  • Seamless integration with existing healthcare systems.
  • User-friendly interface for healthcare professionals.
