King Ferry Presbyterian Church - Web App

This web application was developed for King Ferry Presbyterian Church to streamline administrative tasks and enhance community engagement. The app provides a central hub for events, announcements, and communication among members.

Features:

  • Interactive calendar for events and meetings.
  • Communication tools for members and administrators.
  • Customizable dashboard for community updates.
